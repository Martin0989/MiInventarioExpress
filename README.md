## Descripción

MiInventarioExpress es una aplicación web desarrollada con Node.js, Express, MongoDB y Handlebars para la gestión de productos.

La aplicación permitirá administrar productos, manejar autenticación de usuarios, cargar imágenes y utilizar un chat en tiempo real entre usuarios autenticados.

## Tecnologías utilizadas

- Node.js
- Express
- MongoDB
- Mongoose
- Handlebars
- Multer
- Express-session
- Bcrypt
- Express-validator
- Socket.io
